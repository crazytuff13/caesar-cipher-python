# Secret Agent Caesar Cipher

A full-stack web application that combines classic Caesar Cipher 
encryption with an AI-powered chatbot. Built and deployed as a 
live web app.

 **Live Site:** [Insert your render.com link here]

## Features
-  Encrypt secret messages using Caesar Cipher
-  Decrypt encoded messages
-  AI chatbot powered by Google Gemini
-  Ask the AI questions about cryptography and secret agents
-  Fully deployed and accessible from any browser

## How It Works
Users can type a message, choose a cipher shift number, and 
instantly encrypt or decrypt it. The built-in Gemini AI chatbot 
can answer questions, explain how encryption works, or just 
chat like a secret agent assistant.

## Technologies Used
- Python
- Caesar Cipher Algorithm
- Google Gemini AI API
- HTML/CSS
- Render.com (deployment)
